## 📌 Overview

AI-powered dynamic pricing platform that predicts product demand using machine learning and determines the profit-maximizing price using constrained optimization.

This system simulates how real-world companies optimize pricing to increase revenue.

---

## 🧠 What It Does

1. Predicts demand based on:
   - Product
   - Season
   - Marketing budget
   - Competitor price

2. Calculates profit:
   Profit = (Price − Cost) × Predicted Demand

3. Uses numerical optimization to find the price that maximizes profit.
